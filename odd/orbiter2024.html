Orbiter 2024 Install Checklist

Main Installation

  1. Install Orbiter 2024 (2.67 GB, Forum, Direct)

Mod Installation

169 MB total if excluding UCGO:
  1. Download Altea Aerospace Mods (68.5 MB total, Altea):
    • XR2 (35.0 MB)
    • XR5 (20.8 MB)
    • DG XR1 (12.6 MB)
  2. Download Dan's Mods (Currently only DGIV works in 2024):
    • DeltaGlider IV (86.2 MB, Dan, Direct)
    The Arrow Freighter doesn't 100% work yet but if you want to try it: (360 MB, excl. XR2, Method Source):
    1. UCGO 3.0 + UMmu 3.0 - 2014 (192 MB, Dan, Direct)
    2. OrbiterSound 5.0 (96.2 MB, Dan, Direct)
    3. GeneralVehicle (6.70 MB Forum)
    4. Sound Bridge OS (604 kB, Forum)
    5. XR2 (See above)
    6. Arrow Freighter 2016 (63.5 MB, Direct)
    7. Arrow Freighter 2016 Patch (367 kB, Direct)
    Afterwards, enable both XRSound and OrbiterSound.
  3. Download essential MFDs (14.8 MB total)

      • ModuleMessagingEXT (962 kB, Forum)

    • Download MFDs (13.6 MB total):

Settings Tweaks

  • Options→Visual Settings→ Enable Cloud Shadows, Specular Ripples, Local Light Sources
  • Options→Instruments and Panels→Virtual Cockpit MFD texture size: 1024 x 1024
  • XRSound\XRSound.cfg - line 554: Volume = 0.0
  • Modules: Enable:
    • BaseSyncMFD
    • BurnTimeMFD
    • LaunchMFD
    • TransX
    • DrawOrbits
    • InterMFD57
    • LunarTransferMFD
    • XRSound
    • Framerate
    • ScnEditor
  • Video→Graphics Engine = D3D9 Client, set windowed resolution as appropriate
