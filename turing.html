Turing Machine


Line 0 | Pointer: 0 | State: [PRESS RESET]

  • ADD - add value to pointer
  • AND - returns argument if value at pointer is 0, otherwise, returns value at pointer.
  • DEC - decrement value at pointer
  • DIV - divide value from pointer
  • GOT - move pointer to value at pointer
  • INC - increment value at pointer
  • INP - change value at pointer to state
  • IOR - returns argument if value at pointer is 1, otherwise, returns value at pointer.
  • JIF - jump to line if value at pointer is same as argument
  • JIZ - jump to line if value at pointer is zero
  • JMP - jump to line
  • JNZ - jump to line if value at pointer is not zero
  • MOD - modulo of value to pointer by argument
  • MUL - multiply value to pointer
  • NEG - negates value at pointer
  • NOP - does nothing
  • OUT - change state to target of pointer
  • PNT - change target of pointer
  • SET - change state to argument
  • STO - store value to address
  • SUB - substract value from pointer
