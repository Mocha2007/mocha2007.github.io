Here I add to the lambda functions on this page:

Brief Reminder

  • True := λab.a
  • False := λab.b
  • 0 := λab.b
  • 1 := λab.a b
  • 2 := λab.a (a b)
  • 3 := λab.a (a (a b))
  • etc.

Mathematical Operators

  • LCM := λab. IDIV (MULT a b) (GCD a b)

Booleans

  • NAND := λab. a (NOT b) True
  • NOR := λab. a False (NOT b)
  • XNOR := λab. a (NOT b) b
  • Implication := λab. a b True
  • (Converse Implication) := λab. a a (NOT b)
  • Nonimplication := λab. a (NOT b) a
  • (Converse Nonimplication) := λab. a False b
