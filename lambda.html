Here I add to this page important lambda functions I felt needed:

Brief Reminder

  • True := λab.a
  • False := λab.b
  • 0 := λab.b
  • 1 := λab.a b
  • 2 := λab.a (a b)
  • 3 := λab.a (a (a b))
  • etc.

Mathematical Operators

  • LCM := λab. IDIV (MULT a b) (GCD a b)
  • NCR := λab. IDIV (NPR a b) (FACTORIAL b)
  • NPR := λab. IDIV (FACTORIAL a) (FACTORIAL (SUB a b))
  • TRIANGULAR := λa. PLUS ((EXP a 2) a) 2
  • WOODALL := λa. PRED (MULT a (POW 2 a))
The latter two aren't particularly important but I found them to be fairly simple to implement, so I did so anyways.

Booleans

  • NAND := λab. a (NOT b) True
  • NOR := λab. a False (NOT b)
  • XNOR := λab. a (NOT b) b
  • Implication := λab. a b True
  • (Converse Implication) := λab. a a (NOT b)
  • Nonimplication := λab. a (NOT b) a
  • (Converse Nonimplication) := λab. a False b
