Bounds

Bounds, for proofs (squeeze theorem? comparison tests?). Most of these are derived from various taylor expansions.

$$ \forall x \in \{x \mid 0 < x\} : -\frac{1}{6}x^3 + x < \sin(x) < x$$ $$ \forall x \in \mathbf{R} : -\frac{1}{2}x^2 + 1 ≤ \cos(x) ≤ 1$$ $$ \forall x \in \{x \mid 0 < x < 1 \} : -\frac{1}{2}x^2 + x < \cos(x) < -\frac{4}{9}x^2 + x$$ $$ \forall x \in \{x \mid 0 < x < \frac{\pi}{2}\} : x < \tan(x)$$ $$ \forall x \in \{x \mid 0 < x < 1\} : x < \tan(x) < x^2+x$$ $$ \forall x \in \{x \mid 0 < x < 1\} : \frac{1}{6}x^3 + x < \arcsin(x) < \frac{4}{7}x^3 + x$$ $$ \forall x \in \{x \mid 0 < x < 1\} : -\frac{\pi}{2}x + \frac{\pi}{2} < \arccos(x) < -\frac{1}{6}x^3 - x + \frac{\pi}{2}$$ $$ \forall x \in \mathbf{R} : -\frac{1}{4}x^2+x ≤ \arctan(x) ≤ \frac{1}{4}x^2+x$$ $$ \forall x \in \{x \mid 0 < x\} : -\frac{1}{4}x^2+x < \arctan(x) < x$$ $$ \forall x \in \mathbf{R} : \frac{1}{6}x^3 + \frac{1}{2}x^2 + x + 1 ≤ \exp(x)$$ $$ \forall x \in \{x \mid 0 < x < 1\} : \frac{1}{6}x^3 + \frac{1}{2}x^2 + x + 1 < \exp(x) < \frac{3}{4}x^2 + x + 1$$ $$ \forall x \in \{x \mid 0 < x ≤ 2\} : \frac{1}{2}x^2 - \frac{3}{5}x + 1 < x! < \frac{5}{9}x^2 - \frac{1}{2}x + 1$$
